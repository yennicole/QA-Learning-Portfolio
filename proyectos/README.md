# Proyectos

Proyectos integradores y scripts útiles que combinan múltiples conceptos.

## Proyectos Completados

### 1. Calculadora Básica
**Archivo:** `calculadora.py`  
**Descripción:** Calculadora simple que realiza operaciones básicas (suma, resta, multiplicación, división)  
**Conceptos aplicados:** Variables, operadores, condicionales, entrada/salida de usuario

### 2. Validador de Contraseñas
**Archivo:** `validador_passwords.py`  
**Descripción:** Valida si una contraseña cumple requisitos de seguridad (longitud, mayúsculas, números, caracteres especiales)  
**Conceptos aplicados:** Strings, condicionales, funciones

### 3. Generador de Casos de Prueba
**Archivo:** `generador_casos_prueba.py`  
**Descripción:** Genera combinaciones de datos de prueba para formularios  
**Conceptos aplicados:** Listas, loops, funciones, archivos
