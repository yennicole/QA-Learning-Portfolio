# Testing Manual

Documentación de prácticas de testing manual y casos de prueba.

## Casos de Prueba

Aquí documento casos de prueba para diferentes aplicaciones y escenarios.

### Estructura de casos de prueba:
- ID del caso
- Descripción
- Precondiciones
- Pasos a seguir
- Resultado esperado
- Resultado actual
- Estado (Pass/Fail)

## Bug Reports

Ejemplos de reportes de bugs encontrados durante prácticas.

### Formato de bug report:
- Título
- Severidad
- Descripción
- Pasos para reproducir
- Resultado esperado vs actual
- Evidencia (screenshots si aplica)

- `casos-prueba-login.xlsx` - Casos de prueba para formulario de login
- `casos-prueba-registro.xlsx` - Casos de prueba para registro de usuarios
